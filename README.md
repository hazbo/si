# si

si *(/'saÉª/)*, is currently in the very early stages of development.

### Build instructions

```bash
$ git clone git@github.com:myguide/si.git
$ cd si
$ export GOPATH=$(pwd)/vendor
$ make
```

To install: `$ sudo make install`

### Running

```bash
$ si --version
$ si run script.wren
```

### Tests

```bash
$ make test
```
